# This is a sample text file for Erwiz 1.0.4. (Tutorial 02)
#

# 
# Relationships
# 
# A relationship line has following:
#   * Two entity names which are nomally enclosed by square brackets.
#   * A cardinality string which is placed between the entity names.
# 
# A cardinality string has four characters as follows:
#   * The first character is a cardinality symbol for the first entity.
#   * The second character is always "-".
#   * The third character is always "-".
#   * The fourth character is a cardinality symbol for the second entity.
# 
# If both of the two cardinality symbols are "-", it indicates that
# the cardinality is not specified. The other cardinality symbols 
# will be explained in other files.
#

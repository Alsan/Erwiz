The Options Supported by Erviz
Target Option Name Description
Global title The title of the ERD.
title-size The size of title.
link-files The file path or URL to link files with entities. This option is for svg or map.
The "${entity}" in the specified string is replaced to the entity name.
Entity color The color of the entity. (white/red/blue/green/yellow/orange)
mark The arbitrary phrase to mark an entity with.
Attribute mark The arbitrary phrase to mark an attribute with.
Relationship n1 The arbitrary cardinality string for the first entity.
n2 The arbitrary cardinality string for the second entity.
verb-reverse True if the direction of verb phrase should be reversed.

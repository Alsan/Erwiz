What is Erviz?

Erviz is a free tool to draw ERDs[1] quickly. It works on Windows, Unix/Linux, etc.

By using Erviz, you can create ERDs by the following procedures:

  1. The user creates a text file about some entities and relationships.
    The text file is written in the simple language.
  2. The user converts from the text file to the ERD image file by using Erviz. [2]
    The ERD notation and the image type are specified on conversion time.

For more examples, see Samples and Tutorials.

  1. ERD (Entity-Relationship Diagram) is diagram to represent Entity-Relationship model, a kind of data models. It is used in database designing.
  2. Erviz uses Graphviz to generate image files.

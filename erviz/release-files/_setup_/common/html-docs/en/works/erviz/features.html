The Features of Erviz

The features of Ervis are as follows:

  • Text-Based Data Models

    Erviz supports a simple language to describe data models.
    For details, see Samples and Tutorials.

  • Cooperation with Graphviz

    Erviz generates DOT-formatted graphical files for Graphviz.

  • Freeware

    Erviz is charge-free to use.

  • Easy to Setup

    Download the zip file and unzip it, then you can use Erviz at once.
    Note: To install Java and Graphviz, a little more time is needed.

  • ERD Notations

    Erviz supports IE and IDEF1X notations. (IE: Information Engineering by James Martin)
    You can easily switch these notations on generation time.

  • Automatic Arrangement of Entities

    The user doesn't specify the positions of each entities.
    Graphviz arranges them automatically.

  • Image File Formats

    PNG, JPEG, SVG, PDF, etc.
    It depends on Grapnviz.

  • Multi-Platforms

    Windows, Linux, Mac OS, Solaris, etc.
    Erviz works on various operating systems because it's a Java application,

  • Multi-Languages

    English, Japanese, German, Spanish, Chinese, etc.
    Erviz supports UTF-8 encoded text in input/output data files.

  • Open-Source Software

    The New BSD License is applied to Erviz.

# 
# 動詞句
# 
# リレーションシップ行では、動詞句を指定することも可能です。
# 動詞句は山括弧で囲まれ、2番目のエンティティ名の後に記述されます。
# 
# 動詞句は、次に示すように、方向を持つことができます。
#   * 動詞句と右の山括弧の間に「-」が挿入され、「<動詞->」のようになった場合、
#     方向は「1番目のエンティティから2番目のエンティティ」となります。
#   * 左の山括弧と動詞句との間に「-」が挿入され、「<-動詞>」のようになった場合、
#     方向は「2番目のエンティティから1番目のエンティティ」となります。
#

動作環境

Ervizは、以下のソフトウェアを必要とします。

  • Java Runtime Environment (JRE) 6
  • Graphviz - Graph Visualization Software (Version 2.26以降を推奨)

これらのソフトウェアがお使いの環境にインストールされていない場合は、 上記のURLからダウンロードし、インストールしてください。 (もしくは、APTやRPMなどのパッケージ管理システムを使用してください。)

なお、Ervizは、Graphvizを動的にも静的にもリンクしません。 Graphvizパッケージに含まれるdotコマンドを起動するだけです。

Ervizのテキスト (チュートリアル 02)

#
# このファイルはErviz 1.0.4のサンプルテキストです。(Tutorial 02)
#

# 
# リレーションシップ
# 
# リレーションシップ行は以下の内容で構成されます。
#   * 2つのエンティティ名 (通常は、それぞれが角括弧で囲まれています。)
#   * エンティティ名の間に記述されるカーディナリティ文字列
# 
# カーディナリティ文字列は、以下のような4つの文字で構成されます。
#   * 1番目の文字は、1番目のエンティティに対するカーディナリティ記号です。
#   * 2番目の文字は、常に「-」となります。
#   * 3番目の文字は、常に「-」となります。
#   * 4番目の文字は、2番目のエンティティに対するカーディナリティ記号です。
# 
# 2つのカーディナリティ記号が両方とも「-」だった場合、カーディナリティが
# 指定されなかった事を示します。その他のカーディナリティ記号については、
# 他のファイルで説明します。
#

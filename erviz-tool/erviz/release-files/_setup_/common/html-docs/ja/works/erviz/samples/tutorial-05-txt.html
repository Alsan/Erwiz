Ervizのテキスト (チュートリアル 05)

#
# このファイルはErviz 1.0.4のサンプルテキストです。(Tutorial 05)
#

# 
# 従属エンティティ
# 
# リレーションシップが依存リレーションシップの場合、そのリレーションシップを定義する行の中で
# 依存側のエンティティを丸括弧で囲みます。
# 
# エンティティは、以下の条件のどれかに該当する場合に従属エンティティと判定されます。
#   * エンティティ名の行で、エンティティ名が丸括弧で囲まれている。
#   * 主キーかつ外部キーであるような属性を1つ以上持っている。
#   * 1つ以上の依存リレーションシップで、依存側になっている。
# 

{title: "従属エンティティ"}

[商品]
*商品ID
 商品名
 標準価格

[仕入先]
*仕入先ID
 仕入先名

[商品仕入先]
*商品ID*
*仕入先ID*
*枝番
 適用開始日
 適用終了日

[商品] 1--* (商品仕入先)

[仕入先] 1--* (商品仕入先)

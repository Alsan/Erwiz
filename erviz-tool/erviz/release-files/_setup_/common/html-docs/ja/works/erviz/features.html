Ervizの特徴

Ervisには以下のような特徴があります。

  • テキストによるデータモデル記述

    Ervizは、データモデルを記述するための簡単な言語をサポートします。
    詳しい内容は、サンプルとチュートリアルを参照してください。

  • Graphvizとの連携

    Ervizは、Graphvizのグラフファイル(DOT形式)を生成します。

  • フリーウェア

    Ervizは無償で使用できます。

  • セットアップが簡単

    ZIPファイルをダウンロードして解凍すると、すぐに使用できます。
    注: JavaとGraphvizをインストールする場合、少し時間がかかるかもしれません。

  • ER図の表記法

    IE形式およびIDEF1X形式をサポートします。
    これらの表記法は、ER図の生成時に簡単に切り替えられます。

  • エンティティの自動配置

    ユーザによるエンティティ位置の指定は必要ありません。
    配置はGraphvizによって自動的に行われます。

  • 画像ファイルの形式

    PNG、JPEG、SVG、PDFなど。
    利用可能な出力ファイル形式は、Graphvizに依存します。

  • マルチプラットフォーム

    Windows、Linux、Mac OS、Solarisなど。
    ErvizはJavaアプリケーションであるため、様々なOSで動作します。

  • 多言語対応

    英語、日本語、ドイツ語、スペイン語、中国語など。
    Ervizは、入出力データファイルでUTF-8をサポートします。

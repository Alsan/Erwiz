Ervizのテキスト (チュートリアル 01)

#
# このファイルはErviz 1.0.4のサンプルテキストです。(Tutorial 01)
#

# 
# エンティティ
# 
# エンティティの定義はエンティティ名で始まります。エンティティ名は、通常は角括弧で囲まれています。
# エンティティ名を持つ行の後に、属性名の行(複数)を続けることができます。
# 
# 属性行は、キー記号を含むことができます。
#   * 属性名の左側の「*」は、その属性が主キーまたは複合主キーの一部であることを示します。
#   * 属性名の右側の「*」は、その属性が外部キーであることを示します。
# 
# 属性行は、以下のどれかによって終了します。
#   * 空行
#   * 次のエンティティ名の行
#   * リレーションシップ行 (他のサンプルファイルで説明します。)
#   * EOF (ファイル終端)
# 
# 以下の条件を全て満たす場合、エンティティの定義は省略可能です。
#   * リレーションシップの定義の中にエンティティ名が現れる。
#   * エンティティが属性を持たない。
#   * エンティティがオプションを持たない。
#

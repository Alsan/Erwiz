Ervizのオプション一覧
対象 オプション名 説明
全体 title ER図のタイトルを指定します。
title-size タイトルのサイズを指定します。
link-files エンティティにリンクを設定する場合に、ファイルのパスまたはURLを指定します。このオプションは、svg/map用です。
指定した文字列内の「${entity}」は、各エンティティ名に置き換えられます。
エンティティ color エンティティの色を指定します。(white/red/blue/green/yellow/orange)
mark エンティティに印を付けるための任意の語句を指定します。
属性 mark 属性に印を付けるための任意の語句を指定します。
リレーションシップ n1 1番目のエンティティに対する任意のカーディナリティ文字列を指定します。
n2 2番目のエンティティに対する任意のカーディナリティ文字列を指定します。
verb-reverse 動詞句の方向を反転する場合はtrueに設定します。

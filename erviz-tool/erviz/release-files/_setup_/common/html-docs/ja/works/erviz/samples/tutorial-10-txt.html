Ervizのテキスト (チュートリアル 10)

#
# このファイルはErviz 1.0.4のサンプルテキストです。(Tutorial 10)
#

# 
# オプション
# 
# このファイルでは、オプションをいくつか紹介します。
# 
# Ervizサポートするオプションは、以下のようなグループに分類されます。
#   * 全体オプション (ER図全体に適用される)
#   * エンティティオプション (エンティティに適用される)
#   * エンティティ属性オプション (エンティティ属性に適用される)
#   * リレーションシップオプション (リレーションシップ属性に適用される)
# 
# オプションは、波括弧で囲まれ、通常はオプション指定の対象となる要素の後に記述されます。
# オプション指定でのオプション名とオプション値は「:」で区切られます。
# 波括弧のペアの中に複数のオプションを指定する場合は、「;」で区切られます。
# オプション名では、大文字・小文字は区別されません。
# 
# 現在のバージョンでは、Ervizは以下オプションをサポートします。
#   * title
#       全体オプションです。ER図のタイトルを指定します。
#   * title-size
#       全体オプションです。タイトルのサイズを指定します。(デフォルト:12)
#   * color
#       エンティティオプションです。エンティティの色を指定します。
#   * mark
#       エンティティオプションまたはエンティティ属性オプションです。任意の語句を指定します。
#   * N1
#       リレーションシップオプションです。1番目のエンティティに対する任意のカーディナリティ文字列を指定します。
#   * N2
#       リレーションシップオプションです。2番目のエンティティに対する任意のカーディナリティ文字列を指定します。
#   
#   注意：上記の一覧では、一部のオプションは省略されています。
#

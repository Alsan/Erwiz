Independent Entities and Dependent Entities in Erwiz

If an entity satisfy at least one of the following conditions, Erwiz regards the entity as a dependent entity.

  • The entity name is enclosed by round brackets in the text data.
  • The entity has at least one attribute which is both primary key and foreign key.
  • The entity is specified as the dependent entity in at least one identifying relationships.

If an entity is not dependent, it is independent.

Prerequisites

Erwiz needs the following software:

  • Java Runtime Environment (JRE) 6
  • Graphviz - Graph Visualization Software (Version 2.26 or later is recommended)

You can download them from the URLs. (You can also use a package management system like APT, RPM, etc.)

Note that Erwiz doesn't link Graphviz statically or dynamically. It only launches the dot command in Graphviz package.

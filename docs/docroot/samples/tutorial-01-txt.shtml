# 
# Entities
# 
# The definition of an entity starts with the entity name which is normally enclosed by square brackets.
# An entity name line can be followed by attribute name lines.
# 
# An attribute name line can contain key symbols as follows:
#   * The left side "*" of the name indicates that it is a primary key or a part of compound primary key.
#   * The right side "*" of the name indicates that it is a foreign key.
# 
# The attribute name lines are terminated by any of the following:
#   * A blank line
#   * A next entity name
#   * A relationship line (which will be explained in other sample files)
#   * EOF (end of file)
# 
# The definition of an entity is omissible if all of the following contidions are satisfied:
#  * The entity name is appeared in the definition of a relationship.
#  * The entity doesn't have any attributes.
#  * The entity doesn't have any options.
#

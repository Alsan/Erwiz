What is Erwiz?

Erwiz is a free tool to draw ERDs[1] quickly, a fork of the original project Erviz. As Erviz does, also Erwiz works on Windows, Unix and Linux.

The main goal of Erwiz is to provide the user a simpler and more comfortable way to get his text-based entitity-relationship wizardry converted to a raster or vector graphic.

An entity-relationship diagram is easily created by just

  1. writing a text file using a very simple language to define the entities and relationships and
  2. finally calling the erwiz tool erwiz model.txt erd.png . The ERD notation can be specified on conversion time.
